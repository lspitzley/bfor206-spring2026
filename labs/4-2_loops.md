# BFOR 206 Lab 4-2: Loops

## Task Description

Write a script that adds any quantity of numbers together.

The script should use an if statement to check that there is at least one argument.
If there is at least one argument, add them together and show the result.
If there are no arguments, let the user know and
exit the script without trying to add the numbers together.

## Test Cases

```shell
######## Normal scenario ##########

# Input: 
python loops_lab.py 20 21
# Output: 
sum = 41

# Input:  
python loops_lab.py 9
# Output: 
sum = 9

# Input 
python loops_lab.py 1 2 3 9 10 111
# Output: 
sum = 136

####### Abnormal Scenario ########

# Input:
python loops_lab.py
# Output:
Error! No arguments provided.

```
