# BFOR 206 Lab 5-1: Functions

## Task Description

Create a function that returns the sum of two numbers from
the command line.

## Normal Scenario

### Input

**Arguments:** Two numbers

### Output

**Terminal:** The sum of two numbers.

## Test Cases

### Case 1

```bash
# run this in the terminal
python functions_lab.py 8 9
# output
sum = 17

```
